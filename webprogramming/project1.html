Point Fighting Game

Rules

  • Make some friends it a two player game
  • Each player picks a character at the select screen
  • This game is point fighitng style. Meaning at every time a person is hit a point will be scored
  • 2 minute time limit per game
  • Winner either has most points at the end of time
